Matemáticas

Un curso para desarrollar habilidades en álgebra, geometría y estadística en el segundo año de bachillerato.

Primer Parcial

1. Ecuaciones Lineales

Las ecuaciones lineales representan relaciones lineales entre variables. Se resuelven usando operaciones inversas.

Ejemplo: Resolución

2x + 3 = 7
2x = 7 - 3
2x = 4
x = 2

2. Sistemas de Ecuaciones

Los sistemas de ecuaciones resuelven múltiples ecuaciones simultáneamente, usando métodos como sustitución o eliminación.

Ejemplo: Sustitución

x + y = 5
2x - y = 1
Solución: x = 2, y = 3

3. Factorización

La factorización descompone expresiones algebraicas en factores, como diferencia de cuadrados o trinomios.

Ejemplo: Factorización

x² - 4 = (x - 2)(x + 2)

4. Funciones Lineales

Las funciones lineales tienen la forma y = mx + b, donde m es la pendiente y b el intercepto.

Ejemplo: Gráfica

Gráfica y = 2x + 1. Identifica la pendiente (2) y el intercepto (1).

Segundo Parcial

1. Ecuaciones Cuadráticas

Las ecuaciones cuadráticas (ax² + bx + c = 0) se resuelven por factorización, fórmula general o completando el cuadrado.

Ejemplo: Fórmula General

x² + 2x - 3 = 0
x = [-b ± √(b² - 4ac)] / 2a
x = [-2 ± √(4 + 12)] / 2
x = [-2 ± 4] / 2
x = 1, -3

2. Geometría Plana

Estudio de figuras como triángulos, círculos y polígonos, y sus propiedades (área, perímetro).

Ejemplo: Área de un Triángulo

Base = 6 cm, altura = 4 cm. Área = (base × altura) / 2 = 12 cm².

3. Trigonometría Básica

Introducción a seno, coseno y tangente en triángulos rectángulos.

Ejemplo: Seno

En un triángulo rectángulo, seno A = opuesto / hipotenusa. Calcula seno A si opuesto = 3, hipotenusa = 5.

4. Estadística Descriptiva

La estadística descriptiva organiza datos mediante media, mediana, moda y gráficos.

Ejemplo: Media

Datos: 2, 4, 6, 8. Media = (2 + 4 + 6 + 8) / 4 = 5.

Tercer Parcial

1. Funciones Cuadráticas

Las funciones cuadráticas (y = ax² + bx + c) tienen gráficas en forma de parábola.

Ejemplo: Vértice

Para y = x² + 2x + 1, el vértice es x = -b/2a = -2/2 = -1.

2. Geometría Analítica

Estudio de figuras geométricas usando coordenadas, como la ecuación de una recta o una circunferencia.

Ejemplo: Ecuación de la Recta

Puntos (1,2) y (3,6). Pendiente = (6-2)/(3-1) = 2. Ecuación: y = 2x.

3. Probabilidad

La probabilidad mide la posibilidad de que ocurra un evento, usando fracciones o porcentajes.

Ejemplo: Probabilidad

En una bolsa con 3 bolas rojas y 2 azules, P(roja) = 3/5.

4. Sucesiones

Las sucesiones son secuencias de números con un patrón, como aritméticas o geométricas.

Ejemplo: Sucesión Aritmética

2, 5, 8, 11. Diferencia = 3. Término n = 2 + (n-1)·3.
